A comprehensive WordPress plugin for event organizers, venue managers, and ticketing agencies. Features dynamic event pages with tiered ticket pricing and capacity limits, QR code check-in scanning via mobile, promotional code and early bird discount management, seating chart integration, automated attendee email sequences, and post-event feedback surveys. Integrates with Stripe and PayPal for payment processing, and exports attendee manifests in standard formats for venue compliance requirements.

== Description ==

A comprehensive WordPress plugin for event organizers, venue managers, and ticketing agencies. Features dynamic event pages with tiered ticket pricing and capacity limits, QR code check-in scanning via mobile, promotional code and early bird discount management, seating chart integration, automated attendee email sequences, and post-event feedback surveys. Integrates with Stripe and PayPal for payment processing, and exports attendee manifests in standard formats for venue compliance requirements.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > WordPress Event Ticketing and Venue Management Plugin
